# 支持 CodeWiz 会话

## 新增功能

- 可在设置中开启 CodeWiz 会话来源，搜索、筛选、查看并继续本机和 SSH 远程机器上的 CodeWiz 历史会话。
- 支持把会话迁移或恢复到 CodeWiz，并从对应项目中继续对话。
- 新版本提示支持选择立即更新、暂时隐藏，或隐藏当前版本直到下个版本发布；手动检查更新时仍可重新查看。
- CodeWiz 来源标识改为柔和的小红书红，更容易和其他 agent 区分。

## Bug 修复

- 修复远程 CodeWiz 会话时间都显示为同步时间的问题，现在会按真实对话时间排序和展示。
- 修复 CodeWiz 会话用量可能显示为 0 或偏低的问题，现在会统计真实对话中的输入、输出、缓存和思考 token。
- 修复多个来源标签的悬浮提示可能过窄、过长或位置不准的问题，现在只在鼠标悬浮当前标签时显示，并保持更易读的宽度。
- 修复打开历史记录需要双击且容易选中文本的问题，现在单击即可打开详情。
- 没有可用 token 用量数据的会话不再显示 0 token，避免把未知用量误读成真实统计。
- 个性化来源开关、终端偏好等设置会在重启和更新后保持；首次安装默认只展示 Claude Code 和 Codex 来源。
